# Lista de Exercícios — Arrays em Java

## Questão 1

Crie um método que receba um array de números inteiros e exiba todos os elementos na ordem inversa.

## Questão 2

Crie um método que receba um array de números inteiros e retorne um novo array contendo os mesmos valores em ordem invertida.

## Questão 3

Crie um método que receba um array de números inteiros e retorne um novo array contendo apenas os números pares.
Caso não exista nenhum número par, retorne `null`.

---

## Questão 4

Crie um método que receba um array de Strings e uma String de busca.
O método deve retornar a posição onde a String foi encontrada no array.
Caso ela não exista, retorne `-1`.

---

## Questão 5

Crie um método que receba um array de valores booleanos.
O método deve retornar um array de inteiros onde:

- `true` → 1
- `false` → 0

---

## Questão 6

Crie um método que receba dois arrays de números inteiros.
O método deve retornar um novo array contendo todos os elementos dos dois arrays unidos.

---

## Questão 7

Crie um método que receba dois arrays de números inteiros.
O método deve criar um novo array obedecendo às seguintes regras:

- números negativos devem ficar no início;
- números positivos devem ficar no final.

---

## Questão 8

Crie um método que receba um array de caracteres.
O método deve exibir todos os caracteres formando uma palavra ou frase.

---

## Questão 9

Crie um método que receba um array de números inteiros.
O método deve calcular e retornar a média dos valores do array.

---

## Questão 10

Crie um método que receba um array de números inteiros.
O método deve calcular e retornar a média apenas dos números pares.

---

## Questão 11

Crie um método que receba um array de números inteiros e uma posição.
O método deve retornar o valor armazenado naquela posição do array.

---

## Questão 12

Crie um método que receba um array de números reais (`double`).
O método deve verificar se existe pelo menos um número negativo no array.
Retorne:

- `true` caso exista;
- `false` caso contrário.

---

## Questão 13

Crie um método que receba um array de números inteiros.
O método deve retornar o maior valor presente no array.

---

## Questão 14

Crie um método que receba um array de números inteiros.
O método deve retornar a posição do maior elemento do array.

---

## Questão 15

Crie um método que receba um array de números inteiros.
O método deve retornar o menor valor presente no array.

---

## Questão 16

Crie um método que receba um array de números inteiros.
O método deve retornar a posição do menor elemento do array.

---

## Questão 17

Crie um método que receba dois arrays de números inteiros.
O método deve verificar se os arrays são iguais.
Dois arrays serão considerados iguais quando:

- possuírem o mesmo tamanho;
- possuírem os mesmos valores nas mesmas posições.

---

## Questão 18

Crie um método que receba um array de caracteres.
O método deve converter os caracteres em uma String e retorná-la.

---

## Questão 19

Crie um método que receba um número inteiro representando o tamanho de um array.
O método deve:

1. criar um array de Strings;
2. solicitar que o usuário digite os valores;
3. retornar o array preenchido.

---

## Questão 20

Crie um método que receba:

- um array de Strings;
- uma posição do array.
  O método deve remover o elemento daquela posição, deslocando os demais elementos para a esquerda.
  A última posição deverá receber `null`.

---

## Questão 21

Crie um método que receba um array de números reais (`double`).
O método deve exibir apenas os elementos que estão em posições pares do array.
